5.Show the different selectors and how they are accessed

CSS id Selector

he id selector uses the id attribute of an HTML element to select a specific element. The id of an element is unique within a page, so the id selector is used to select one unique element! To select an element with a specific id, write a hash (#) character, followed by the id of the element

CSS Class Selector

The class selector selects HTML elements with a specific class attribute. To select elements with a specific class, write a period (.) character, followed by the class name.

The CSS Universal Selector

The universal selector (*) selects all HTML elements on the page.
